Description:
    The Logical Bank Manager is a web app for customers to link their bank accounts and arrange their money in “partitions”.
    Partitions contain customization rules to give the user control over their finacial management.

Authors:
    Jonathan Bogie
    Carston Konyn

    Kyle Pinkham

Created: 11/27/23

Course: Software Methodology I (CS 422)

Assignment: Project

Steps to compile:
    cd LBM
    ./runserver.sh
    open the link

Dependencies: 
    Python3
    Django

Structure:
    LBM:
        Contains the views and models for the LBM web app. Includes login, partitions, users, etc.
    DummyBank:
        Mimic data we would recieve from a user's bank API
